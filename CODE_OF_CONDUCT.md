# Code of Conduct

## Our Standard

This repository is used for distributing and maintaining the AJ Tools installer for Autodesk Revit.

Participation in this repository is expected to remain respectful, technical, and relevant to the project. Contributors, users, and maintainers should:

- communicate clearly and professionally
- focus on reproducible technical issues and concrete improvements
- respect differing experience levels and project constraints
- avoid harassment, insults, or disruptive behavior

## Unacceptable Behavior

The following behavior is not acceptable in issues, pull requests, reviews, or other project interactions:

- personal attacks, harassment, or discriminatory language
- spam, trolling, or repeated off-topic comments
- sharing private, confidential, or proprietary information without permission
- aggressive or intentionally unconstructive conduct

## Scope

This Code of Conduct applies to all public interactions in this repository, including:

- issues
- pull requests
- code review comments
- release discussions and related communication

## Maintainer Responsibilities

Maintainers may edit, hide, lock, or remove comments, issues, or pull requests that violate this Code of Conduct. Repeated or serious violations may result in being blocked from further participation in this repository.

## Reporting

If you encounter behavior that violates this Code of Conduct, open a private report by emailing `ajmalnattika@gmail.com` with:

- a link to the relevant GitHub page
- a short description of the behavior
- any context needed to understand the issue

## Enforcement

Reports will be reviewed and handled case by case. The response will be based on project safety, impact, and the need to keep this repository useful for installation and support work.
