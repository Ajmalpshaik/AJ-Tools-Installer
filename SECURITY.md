# Security Policy

## Safe Downloads

- Download installer packages only from the official GitHub Releases page.
- Verify the package hash against `SHA256SUMS.txt` before installing.
- Do not trust copies of the installer shared outside the official repository.

## Reporting

Report suspicious packages, checksum mismatches, or installer security concerns privately to:

- `ajmalnattika@gmail.com`

## Privacy

- Do not post private project data, customer files, or license information in issues.
- Remove user names and local file-system paths from screenshots or logs before sharing them.
