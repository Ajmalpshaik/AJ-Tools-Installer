# Support

Use this repository for public installer distribution support.

## Use AJ-Tools-Installer For

- release downloads
- checksum verification
- install, uninstall, and upgrade issues
- current-user versus all-users installer questions

Before opening an issue:

- confirm the package came from the official GitHub Releases page
- include the AJ Tools package version, Revit version, and Windows version
- state whether you used current-user or all-users installation
- sanitize screenshots and logs before posting them publicly

## Use AJ Tools For Source Issues

- feature requests
- command behavior defects
- UI or workflow bugs in the add-in itself

Open source issues in [AJ Tools](https://github.com/Ajmalpshaik/AJ-Tools/issues).

## Security

Do not post suspicious-package details, sensitive logs, or customer data publicly.
Follow [SECURITY.md](SECURITY.md) for private reporting instructions.
